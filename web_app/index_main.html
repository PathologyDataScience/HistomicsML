HistomicsML

Interactive machine-learning for histology images.

Region

Region based active learning with convolutional autoencoders.

Nuclei

Cell based active learning with convolutional autoencoders.

Nuclei (segmentation)

Cell based active learning using traditional nuclear segmentation and feature extraction.
